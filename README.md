## 📂Como baixar o projeto

````bash
# Clonar o repositório
$ git clone https://github.com/gabriel-elesbao/podcastr-NLW5.git

# Entrar no diretório 
$ cd contador-de-voltas

#instalar as dependências 
$ yarn install

#rodar  aplicação
$ yarn start 

````
<h3> O app estará disponivel no seu navegador em <h3> 

`http://localhost:3000`
